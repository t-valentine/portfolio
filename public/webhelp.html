Learn HTML & CSS

I promise HTML and CSS aren't hard to learn and if you've been online for a while, you probably already know more than you realize! These are loosely ordered from least technical to most techincal, but any of the 3 are good starting points! W3Schools is an awesome resource and you can learn something at any knowledge level.
